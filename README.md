# Quiz Generator
This project enables users to easily generate custom quizzes, complete them interactively, and review detailed performance results. It's designed to support both learning and self-assessment in an intuitive, user-friendly environment.

## Features
- 📝 Create custom quizzes with multiple-choice questions using a simple text-based format
- 🧠 Take quizzes interactively with immediate feedback and scoring
- 📱 Responsive design for seamless use on desktop and mobile
- 🎨 Modern UI for an intuitive user experience
- 📊 Score analysis to review your performance and identify areas for improvement

### Usage
1. Launch the application in your browser.  
2. Paste or write quiz content using the supported format:
   ```text
   @Q: What is the capital of France?  
   @A: Madrid  
   @A*: Paris  
   @A: Berlin  
   @A: Rome  
   ```
3. Click Generate Quiz to begin.
4. Answer questions and receive your results at the end.
5. Optionally restart or create a new quiz.
